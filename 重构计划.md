# ECHO 重构计划

## 现状问题
1. DashboardView.vue 945 行，职责过多
2. AI 对话使用 mock 数据，未接入真实 LLM
3. 对话历史不持久化
4. 无画像测评页面
5. 知识库弹窗与画像详情混淆

## 重构方案

### 第一阶段：后端完善
- [ ] ChatService 接入真实 AI 对话（替换 generateMockReply）
- [ ] 对话历史持久化（ChatMessage 表）
- [ ] 画像测评接口完善

### 第二阶段：前端拆分
- [ ] 删除 DashboardView.vue（945 行）
- [ ] 创建 HomeView.vue（测评引导 + 画像概览）
- [ ] 创建 ChatView.vue（对话页面，含聊天历史）
- [ ] 创建 PersonaView.vue（画像详情 + 编辑）
- [ ] 创建 QuizView.vue（题库测评）

### 第三阶段：功能集成
- [ ] 对话时自动调用 AI 接口
- [ ] 切换聊天框加载历史消息
- [ ] 测评后自动更新画像
- [ ] 画像详情展示维度分布

## 设计原则
- 每个组件 < 200 行
- 单一职责
- 前后端分离
- 易于维护
